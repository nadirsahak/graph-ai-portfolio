How I will apply it

I will translate course methods into practical, auditable workflows for surveillance and preparedness. This includes structured technical writing, automated analysis and reporting, and responsible task automation with clear human oversight.

Automated situation reporting Surveillance dashboards and QA Literature monitoring workflows Template driven technical writing Reproducible analysis

Modules at a glance

Module 1: Websites, documents and presentations

  • How LLMs work and how to generate outputs responsibly
  • Markdown, LaTeX, and HTML formatting workflows
  • Building simple research websites using no code tooling

Module 2: AI automated data analysis

  • Rapid data analysis using code interpreter style tools
  • Automated dashboards and recurring spreadsheet workflows
  • Quality checks and ethical reporting practices

Module 3: GenAI for research and text analysis

  • Efficient literature search and synthesis systems
  • Text classification workflows and summarization
  • Knowledge maps and structured evidence tables

Module 4: Custom GPTs and chatbots

  • Context management and retrieval concepts
  • Custom assistants for scoped use cases
  • Evaluation and guardrails

Module 5: AI for task automation

  • Agent based workflows and app to app automations
  • Routine writing and response workflows with oversight
  • Browser assisted workflows for structured tasks

Module 6: Communication and outreach

  • Audience specific summaries with transparency
  • Figures, diagrams, and visualization prompting
  • Content creation with integrity

Module 7: Capstone project and showcase

  • Peer feedback and iteration
  • Final presentations and public Q and A session
